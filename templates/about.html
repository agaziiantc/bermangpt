BermanGPT7.0

The Bermanic Text Generator

How do to use this

Just press the generate a bermanic text button! The website will do everything else for you!

How it works

BermanGPT7.0 is an LTSM model that has been trained on a lot of quotes from Dr. Berman's class using PyTorch. When you press "generate" the website activates the model and makes it generate a text based off a randomly generated prompt.

How it's hosted

This website is hosted off my oracle server and uses ngrok port forwarding in order to be accessible from anywhere. This is also why the link is so weird.

Source code

This website is open source and you can find all the code at https://github.com/agaziiantc/bermangpt.

Why is BermanGPT a thing and why is it on iteration 7

My english teacher did not let me write a research paper on birds being government drones for the ISearch project, so I chose the next best thing of making an AI based off my global history teacher. I increased the number by 1 every time I had to remake it because something went horribly wrong (like accidentally nuking the server it's hosted on off the face of the planet or somehow making it no longer have an ip). Also I trained the model until it had a cross-entropy of 7.
